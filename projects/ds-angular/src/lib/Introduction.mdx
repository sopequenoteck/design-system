# DS-Angular

Bibliothèque de composants Angular 20 accessible et thématisable.

## Installation

```bash
npm install @kksdev/ds-angular @angular/cdk @fortawesome/angular-fontawesome @fortawesome/free-solid-svg-icons
```

## Configuration

**1. Importer les styles** dans `src/styles.scss` :

```scss
@use '@kksdev/ds-angular/styles';
```

**2. Activer un thème** dans `app.component.ts` :

```typescript
document.documentElement.className = 'theme-light'; // ou 'theme-dark'
```

## Exemple

```typescript
import { Component } from '@angular/core';
import { DsButton, DsInputField } from '@kksdev/ds-angular';

@Component({
  standalone: true,
  imports: [DsButton, DsInputField],
  template: `
    <ds-input-field label="Email" placeholder="email@exemple.com" />
    <ds-button variant="primary" (clicked)="submit()">Envoyer</ds-button>
  `
})
export class MyComponent {
  submit() { console.log('Envoyé !'); }
}
```

## Composants v1.6.0

La version 1.6.0 introduit 3 nouveaux composants :

### DsInputNumber

Champ de saisie numérique avec boutons d'incrémentation/décrémentation :

```typescript
import { DsInputNumber } from '@kksdev/ds-angular';

@Component({
  standalone: true,
  imports: [DsInputNumber],
  template: `
    <ds-input-number
      [(ngModel)]="quantity"
      [min]="0"
      [max]="100"
      [step]="1"
      size="md"
      label="Quantité"
    />
  `
})
export class MyComponent {
  quantity = 0;
}
```

### DsSegmentedControl

Groupe de boutons radio visuels pour sélection exclusive :

```typescript
import { DsSegmentedControl } from '@kksdev/ds-angular';

@Component({
  standalone: true,
  imports: [DsSegmentedControl],
  template: `
    <ds-segmented-control
      [(ngModel)]="view"
      [options]="viewOptions"
      size="md"
    />
  `
})
export class MyComponent {
  view = 'grid';
  viewOptions = [
    { value: 'grid', label: 'Grille', icon: 'grid' },
    { value: 'list', label: 'Liste', icon: 'list' },
    { value: 'table', label: 'Tableau', icon: 'table' }
  ];
}
```

### DsColorPicker

Sélecteur de couleur complet avec palette, sliders et presets :

```typescript
import { DsColorPicker } from '@kksdev/ds-angular';

@Component({
  standalone: true,
  imports: [DsColorPicker],
  template: `
    <ds-color-picker
      [(ngModel)]="color"
      [showAlpha]="true"
      [presets]="colorPresets"
      size="md"
    />
  `
})
export class MyComponent {
  color = '#3b82f6';
  colorPresets = ['#3b82f6', '#10b981', '#f59e0b', '#ef4444', '#8b5cf6'];
}
```
