## Utilisation

### Dans vos styles SCSS

```scss
.my-component {
  background: var(--background-main);
  color: var(--text-default);
  padding: var(--space-4);
  border-radius: var(--radius-2);
  box-shadow: var(--shadow-1);
}
```

### Avec des fallbacks

```scss
.my-component {
  // Fallback pour compatibilit√©
  color: var(--text-default, #1f2937);
  background: var(--background-main, #ffffff);
}
```

### Media queries avec breakpoints

```scss
@media (min-width: 768px) {
  .my-component {
    padding: var(--space-6);
  }
}
```
