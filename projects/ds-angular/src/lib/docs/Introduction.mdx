# DS-Angular

Bibliothèque de composants Angular moderne, accessible et entièrement thématisable.

---

## Pourquoi DS-Angular ?

<div className="ds-feature-grid">

| Caractéristique | Détail |
|-----------------|--------|
| **47 composants** | Formulaires, navigation, feedback, overlays, data display |
| **Accessibilité WCAG 2.1 AA** | Navigation clavier complète, rôles ARIA, focus management |
| **3 thèmes** | Light, Dark, Custom - avec 100+ tokens CSS personnalisables |
| **Angular 20** | Standalone components, Signals, ControlValueAccessor |
| **Tree-shaking optimisé** | Import uniquement ce dont vous avez besoin (~150KB gzipped total) |
| **TypeScript strict** | Types complets pour l'autocomplétion IDE |

</div>

## Architecture

DS-Angular s'articule autour de **3 couches** :

```
┌─────────────────────────────────────────────────────────┐
│                    VOTRE APPLICATION                     │
├─────────────────────────────────────────────────────────┤
│   47 Composants DS (DsButton, DsModal, DsTable...)      │
│   ├── Logique métier                                    │
│   ├── ControlValueAccessor (formulaires)                │
│   └── CDK Overlay (popups, modals)                      │
├─────────────────────────────────────────────────────────┤
│   7 Primitives (PrimitiveButton, PrimitiveInput...)     │
│   └── Briques atomiques stylisées via CSS custom props  │
├─────────────────────────────────────────────────────────┤
│   Tokens CSS (100+ variables)                           │
│   ├── Couleurs, espacements, typographie                │
│   └── 3 thèmes : Light / Dark / Custom                  │
└─────────────────────────────────────────────────────────┘
```

### Les couches en détail

| Couche | Fichiers | Rôle |
|--------|----------|------|
| **Primitives** | `primitive-button`, `primitive-input`, etc. | Éléments HTML stylisés sans logique, base pour les composants DS |
| **Composants DS** | `ds-button`, `ds-modal`, `ds-table`, etc. | Composants complets avec états, validation, accessibilité |
| **Tokens** | `_tokens.scss`, `_light.scss`, `_dark.scss` | Variables CSS pour personnalisation globale |

## Installation rapide

### 1. Installer le package

```bash
npm install @kksdev/ds-angular @angular/cdk @fortawesome/angular-fontawesome @fortawesome/free-solid-svg-icons
```

### 2. Importer les styles

Dans `src/styles.scss` :

```scss
@use '@kksdev/ds-angular/styles';
```

### 3. Activer un thème

Dans votre `app.component.ts` :

```typescript
import { Component, OnInit } from '@angular/core';

@Component({
  selector: 'app-root',
  standalone: true,
  template: `<router-outlet />`
})
export class AppComponent implements OnInit {
  ngOnInit() {
    // Choisir le thème au démarrage
    document.documentElement.className = 'theme-light';
    // Ou 'theme-dark' pour le mode sombre
  }
}
```

### 4. Utiliser un composant

```typescript
import { Component } from '@angular/core';
import { DsButton, DsInputField } from '@kksdev/ds-angular';

@Component({
  standalone: true,
  imports: [DsButton, DsInputField],
  template: `
    <ds-input-field
      label="Votre email"
      placeholder="email@exemple.com"
      [(ngModel)]="email"
    />
    <button ds-button variant="primary" (click)="submit()">
      Envoyer
    </button>
  `
})
export class ContactFormComponent {
  email = '';

  submit() {
    console.log('Email envoyé:', this.email);
  }
}
```

## Exemples de composants

### Boutons

```html
<button ds-button variant="primary">Principal</button>
<button ds-button variant="secondary">Secondaire</button>
<button ds-button variant="outline">Contour</button>
<button ds-button variant="ghost">Ghost</button>
<button ds-button variant="danger">Danger</button>
```

### Formulaire avec validation

```typescript
import { FormGroup, FormControl, Validators, ReactiveFormsModule } from '@angular/forms';
import { DsInputField, DsSelect, DsButton } from '@kksdev/ds-angular';

@Component({
  standalone: true,
  imports: [ReactiveFormsModule, DsInputField, DsSelect, DsButton],
  template: `
    <form [formGroup]="form" (ngSubmit)="onSubmit()">
      <ds-input-field
        formControlName="name"
        label="Nom complet"
        [state]="form.get('name')?.invalid && form.get('name')?.touched ? 'error' : 'default'"
        errorText="Le nom est requis"
      />

      <ds-select
        formControlName="country"
        label="Pays"
        [options]="countries"
        placeholder="Sélectionnez..."
      />

      <button ds-button
              variant="primary"
              type="submit"
              [disabled]="form.invalid">
        Valider
      </button>
    </form>
  `
})
export class MyFormComponent {
  form = new FormGroup({
    name: new FormControl('', Validators.required),
    country: new FormControl('')
  });

  countries = [
    { value: 'fr', label: 'France' },
    { value: 'be', label: 'Belgique' },
    { value: 'ch', label: 'Suisse' }
  ];

  onSubmit() {
    if (this.form.valid) {
      console.log(this.form.value);
    }
  }
}
```

### Modal avec contenu

```typescript
import { DsModal, DsButton } from '@kksdev/ds-angular';

@Component({
  standalone: true,
  imports: [DsModal, DsButton],
  template: `
    <button ds-button (click)="showModal = true">Ouvrir</button>

    <ds-modal
      [isOpen]="showModal"
      title="Confirmation"
      (closed)="showModal = false"
    >
      <p>Êtes-vous sûr de vouloir continuer ?</p>

      <div slot="footer">
        <button ds-button variant="ghost" (click)="showModal = false">
          Annuler
        </button>
        <button ds-button variant="primary" (click)="confirm()">
          Confirmer
        </button>
      </div>
    </ds-modal>
  `
})
export class ConfirmDialogComponent {
  showModal = false;

  confirm() {
    console.log('Confirmé !');
    this.showModal = false;
  }
}
```

## Nouveautés v1.6.0

### DsInputNumber

Champ numérique avec boutons +/- :

```html
<ds-input-number
  [(ngModel)]="quantity"
  [min]="0"
  [max]="100"
  [step]="1"
  label="Quantité"
/>
```

### DsSegmentedControl

Sélection visuelle exclusive :

```html
<ds-segmented-control
  [(ngModel)]="view"
  [options]="[
    { value: 'grid', label: 'Grille' },
    { value: 'list', label: 'Liste' }
  ]"
/>
```

### DsColorPicker

Sélecteur de couleur complet :

```html
<ds-color-picker
  [(ngModel)]="color"
  [showAlpha]="true"
  [presets]="['#3b82f6', '#10b981', '#f59e0b']"
/>
```
